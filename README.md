# ATC Tuition Management System

## Description
A Java Swing-based system developed to manage tuition centre operations including student registration, class management, payment tracking, and income reporting.

## Features
- Multi-role access for Admin, Receptionist, Tutor, and Student
- Student registration and subject enrollment
- Class and schedule management
- Payment tracking and receipt handling
- Monthly income reporting
- Profile management
- File handling using `.txt` files

## Technologies
- Java
- Java Swing
- NetBeans
- File Handling

## How to Run
1. Open the project in NetBeans
2. Build the project
3. Run `Main.java`

## Project Structure
- `src/` - Java source code
- `data/` - Supporting data files
- `images/` - Images and UI assets
- `nbproject/` - NetBeans project configuration
